# 智能查询

OpenDataWorks 的智能查询模块（DataAgent）基于 AI Agent 实现自然语言到 SQL 的转换，让业务人员无需编写 SQL 即可查询数据。

## 工作原理

```
用户提问 → NL2SQL Agent → 生成 SQL → 执行查询 → 返回结果
```

1. 用户以自然语言描述查询需求
2. AI Agent 理解语义并结合数据库 schema 生成 SQL
3. 系统执行 SQL 并返回结构化结果
4. 支持多轮对话，逐步细化查询

## 核心能力

### 自然语言理解

- 支持中文自然语言输入
- 理解业务术语和数据域概念
- 支持模糊查询和条件组合

### SQL 生成

- 自动识别目标表和字段
- 支持聚合、分组、排序、限制
- 支持多表关联查询
- 生成的 SQL 可审查和编辑

### 多轮对话

- 上下文记忆，支持追问和细化
- 会话管理与历史记录
- 支持在已有结果上追加条件

### 结果展示

- 表格形式展示查询结果
- 支持结果导出

## 使用方式

在前端「智能查询」模块中：

1. 创建新会话或选择已有会话
2. 输入自然语言问题，例如「最近 30 天工作流发布次数趋势」
3. 系统生成并执行 SQL
4. 查看结果，可继续追问

## 技术架构

- **后端**: FastAPI + Python，独立部署为 DataAgent Backend
- **Agent 框架**: Claude Agent SDK
- **会话存储**: MySQL（dataagent schema）
- **任务协调**: Redis
- **通信方式**: SSE 流式响应
