# 工作流编排

OpenDataWorks 集成 Apache DolphinScheduler 实现工作流编排与任务调度。

## 任务类型

### SQL 任务

支持在多种数据源上执行 SQL：

- MySQL
- Apache Doris
- 其他 JDBC 兼容数据源

### Shell 任务

支持执行 Shell 脚本，适用于数据导入导出、文件处理等场景。

## 执行模式

### 单任务执行

快速测试单个任务，不涉及依赖关系：

1. 选择目标任务
2. 点击「执行」
3. 查看执行日志和结果

### 工作流执行

按依赖关系执行完整工作流：

1. 配置任务间的依赖关系
2. 系统自动生成 DAG
3. 同步到 DolphinScheduler
4. 按拓扑顺序执行

## 任务配置

| 配置项 | 说明 |
|--------|------|
| 优先级 | HIGH / MEDIUM / LOW |
| 超时时间 | 任务最大执行时长 |
| 重试次数 | 失败后自动重试次数 |
| 重试间隔 | 两次重试之间的等待时间 |
| 数据源 | 任务执行的目标数据源 |

## DolphinScheduler 同步

系统自动将任务配置同步到 DolphinScheduler：

- 创建任务时自动创建对应工作流
- 修改依赖关系时自动更新 DAG
- 支持工作流上线/下线管理
- 执行状态实时回写

::: tip
DolphinScheduler 连接配置通过前端「系统管理 → Dolphin 配置」维护，无需修改配置文件。
:::
