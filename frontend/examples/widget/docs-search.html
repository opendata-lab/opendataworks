什么是 OpenDataWorks？

OpenDataWorks 是一个面向企业级开发设计的统一数据工作台，融合了元数据管理、工作流编排、数据血缘分析以及 NL2SQL 智能问数服务。

核心组件介绍

平台采用微服务及主从队列架构，提供了一套前后端解耦的数据资产管理系统。 用户不仅可以通过可视化拖拽的形式编辑并发布复杂的 ETL 任务，还可以直接接入智能问数（NL2SQL）小组件，通过自然语言查询底层的业务数据库：

-- 自然语言提问：“最近30天的工作流执行趋势” -- 智能问数小组件将自动转换为以下 SQL 并返回可视化图表： SELECT date(create_time) as date, count(1) as total FROM opendataworks.workflow_task_instance WHERE create_time >= date_sub(now(), interval 30 day) GROUP BY 1 ORDER BY 1;

智能问数 Widget 的引入

如果你想把这一套自然语言查询数据库的能力无缝引入到你们自有的 BI 系统、数据大屏、或是内部业务系统文档中，只需在你的 HTML 网页引入一个 <script> 标签包即可。 它支持轻量的悬浮胶囊形态、侧边栏内嵌形态，或者如本页演示的“搜索对话框式”嵌入。
