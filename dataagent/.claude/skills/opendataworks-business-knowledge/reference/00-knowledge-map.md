# 业务知识地图

先结论：本 Skill 只解析 OpenDataWorks 业务语义，不负责 SQL 方法或工具执行。

## 语义类型到资料

| 需求 | 先看 | 输出 |
| --- | --- | --- |
| 术语解释 | `10-term-index.md` | 术语、别名、歧义、追问建议 |
| 指标口径 | `20-metric-index.md` | 指标公式、默认时间字段、候选映射 |
| 对象本体 | `30-ontology.md` | 业务对象、相关平台表 |
| 业务规则例外 | `40-business-rules.md` | 不能混用的状态、默认过滤、例外规则 |

## 使用原则

- 先判断用户问题是否需要业务知识；如果只是在问 SQL 写法或执行失败，交给通用问数 SQL 方法。
- 如果术语、指标或口径在本 Skill 中有明确定义，优先使用本 Skill 的定义。
- 如果本 Skill 没有定义租户私有术语或默认过滤，不要猜测；给出最小追问。
- 如果语义已经明确，需要查询真实数据时，回到通用问数 SQL skill 或可用的真实工具。
